## Installation

```
1. Download the repository
2. Extract the zip file
3. Open the extracted folder
4. Copy the plugins you want to use to the plugins folder of your BetterDiscord installation
5. Restart Discord
6. Enable the plugins you want to use in the BetterDiscord plugins settings

```

## Code Details
```
1. AppNotifications
   - InAppNotifications.plugin.js
     - Show notifications in the app instead of the system
2. AvatarHover
    - AvatarHover.plugin.js
      - Hover over a user's avatar to see their status
3. BiggerStreamPreview
    - BiggerStreamPreview.plugin.js
      - Make the stream preview bigger
4. CollapsibleUI.
    - CollapsibleUI.config.json
    - CollapsibleUI.plugin.js
      - Make the UI collapsible
5. FriendNotifications.
    - FriendNotifications.config.json
    - FriendNotifications.plugin.js
      - Get notifications when a friend goes online
6. GameActivityToggle.
    - GameActivityToggle.config.json
    - GameActivityToggle.plugin.js
      - Toggle game activity
7. InvisibleTyping.
    - InvisibleTyping.config.json
    - InvisibleTyping.plugin.js
      - Appear offline while typing
8. MessageLoggerV2.
    - MessageLoggerV2.config.json
    - MessageLoggerV2.plugin.js
      - Log messages
9. PlatformIndicator
    - APlatformIndicators.plugin.js
    - PlatformIndicators.config.json
      - Show the platform of users
10. ReadAllNotificationsButton.
    - ReadAllNotificationsButton.config.json
    - ReadAllNotificationsButton.plugin.js
      - Add a button to read all notifications
11. ServerDetails.
    - ServerDetails.config.json
    - ServerDetails.plugin.js
      - Show server details
12. ShowHiddenChannels.
    - ShowHiddenChannels.config.json
    - ShowHiddenChannels.plugin.js
    - ShowHiddenChannelsReturns.config.json
      - Show hidden channels
13. SpotifyControls.
    - SpotifyControls.config.json
    - SpotifyControls.plugin.js
      - Add Spotify controls
14. ViewProfilePicture.
    - ViewProfilePicture.config.json
    - ViewProfilePicture.plugin.js
      - View profile pictures in full size
15. important !!
    > Libraries for plugins installation is mandatory
```
